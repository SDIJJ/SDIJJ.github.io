IntelliJ IDEA的高效使用

开发工具
  • java

读书,健身,努力工作,关心身体,保持好心情,成为最棒的自己!

IEDA高效使用

1.无处不在的跳转

  • 项目之间跳转: Ctrl + Alt + [ / ]
  • 文件之间跳转: Ctrl + E (个人更喜欢: Ctrl + Tab)
  • 跳转已经打开的文件:Ctrl + Alt + 左/右

2.高校定位代码

  • 新建书签 ：F11 (带有数字的书签:Ctrl + F11)
  • 添加收藏:Shift + Alt + F

3.精准搜索

  • 搜索指定类:Ctrl + N
  • 搜索指定文件:Shift + Ctrl + N
  • 寻找指定函数:Shift + Ctrl + Alt +N
  • 搜索指定的字符串: Ctrl + Shift + F

4.列操作

  • 选中其他一样的符号:Shift + Ctrl + Alt + J

5.Postfix

  • 快速写for循环: 100.fori
  • 快速输出: .sout
  • 构造器中快速生成属性: 属性名字.field
  • 快速返回: 返回值.retu..
  • 判断值是否不等于null: 值.null..

6.Alt + Enter

  • 快速创建没有的函数

  • 整理代码☞最简洁状态

  • 快速格式化代码串

  • 快速实现接口

  • 错误单词拼写提示(错的单词下方会有浪线提示)

  • 导包

7.重构

  • 重构变量: Shift + F6
  • 重构函数:
    • 重构参数: Ctrl + F6
    • 直接填加参数修改方法:Alt + Enter(优)

8.抽取

  • 抽取字符串成变量: Ctrl + Alt + V(与有道词典快捷键重复,非常纠结)
  • 抽取成静态常量: Ctrl + Alt + C
  • 抽取成成员变量: Ctrl + Alt + F
  • 抽取函数: Ctrl + Alt + M

9.git的集成

  • 右击这行代码前方点击第一个显示作者信息
  • 查看与版本库中修改过的部分 : Ctrl + Shift + Alt + 上/下
  • 快速撤销:Alt + Ctrl + Z (QQ 冲突)

10 .local history

  • 11.关联一切

  • Spring :直达使用的类

  • 关联上数据库:书写SQL提示

12.断点调试

  • 添加断点:Ctrl + F8
  • 断点运行:
  • 单步运行: F8
  • 下一个断点: F9
  • 查看所有断点:Shift + Ctrl + F8
  • 禁止所有断点:
  • 条件断点: Shift + Ctrl + F8
  • setValue: 位置按F2

13.其他操作

  • 复制文件: F5
  • 移动文件: F6
  • 复制完整文件名: Shift + Ctrl + C
  • 调出剪切板: Ctrl + Shift + V

14 .

  • 查看类的大纲: Ctrl +F12

  • 查看继承图:Ctrl + Shift + Alt + U

  • 查看类的继承关系: Ctrl + H

  • 方法调用层次:Ctrl + Alt + H
